# BELIVE - Rooms near you!
Use a mobile app to find rooms to stay in the night or even buy it!

## Prerequisites
- Flutter 3.24.4
- Dart 3.5.3
- Firebase

## Installation
- Clone this repo
- In the project directory, run `flutter pub get`

## How to run
- use the `flutter run` command on your active device
